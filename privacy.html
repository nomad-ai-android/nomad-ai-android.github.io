개인정보처리방침

최종 업데이트: 2026년 4월 17일 · 시행일: 2026년 4월 17일

NOMAD AI(이하 "앱")는 이용자의 개인정보를 소중하게 다루며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 이용자의 정보를 어떻게 수집, 이용, 저장, 보호하는지 안내합니다.

핵심 요약: NOMAD AI 는 온디바이스 LLM 기반 여행 도우미로, 대부분의 처리가 이용자의 기기 내부에서 이루어집니다. 대화 내용, 카메라 이미지, 경비 기록은 외부 서버로 전송되지 않습니다.

1. 수집하는 개인정보 항목

NOMAD AI 는 회원가입, 로그인, 계정 식별 없이 사용 가능한 앱이며, 이용자로부터 개인 식별 정보를 직접 수집하지 않습니다. 다만, 앱의 기능 제공을 위해 아래와 같은 데이터가 기기 내부에서 처리됩니다.

항목 처리 위치 목적 보관 기간
대화(채팅) 기록 기기 내부 저장소 대화 이어가기, 기능 제공 이용자가 삭제하거나 앱을 삭제할 때까지
카메라 이미지(메뉴 OCR) 기기 내부 메모리 메뉴 텍스트 인식 및 번역 처리 즉시 메모리에서 소멸(저장되지 않음)
경비 기록 기기 내부 저장소 지출 자동 정리 및 통계 이용자가 삭제하거나 앱을 삭제할 때까지
앱 설정(언어, 모델 선택 등) 기기 내부 저장소 사용자 환경 유지 앱 삭제 시까지
네트워크 요청 로그 일시적(휘발성) 환율 조회, AI 모델 다운로드 요청 종료 시 삭제

수집하지 않는 정보

  • 이름, 전화번호, 이메일, 주소 등 개인 식별 정보
  • 주민등록번호 등 고유 식별 정보
  • 계정 정보, 비밀번호
  • 위치 정보(GPS)
  • 연락처, 통화 기록, SMS
  • 광고 식별자(ADID) 및 기기 지문

2. 개인정보의 수집 및 이용 목적

  • 오프라인 AI 대화 기능 제공
  • 메뉴 OCR 및 번역 기능 제공
  • 경비 자동 기록 및 통계 제공
  • 실시간 환율 조회 기능 제공
  • 이용자 환경 및 선호도 유지(앱 설정)

3. 개인정보의 처리 및 보관

NOMAD AI 는 이용자의 대화, 이미지, 경비 데이터를 당사 서버나 제3자 서버로 전송하지 않습니다. 모든 처리는 이용자의 기기 내부에서 이루어지며, 기기 저장소에만 보관됩니다.

  • 데이터 보관 위치: 이용자의 Android 기기 내부 저장소
  • 서버 보관: 없음
  • 자동 삭제: 앱 삭제 시 기기에 저장된 데이터도 모두 함께 삭제됩니다.

4. 외부 전송 및 제3자 제공

앱은 원칙적으로 이용자의 개인정보를 외부로 전송하거나 제3자에게 제공하지 않습니다. 다만, 다음 기능에 한해 최소한의 네트워크 요청이 발생합니다.

요청 대상 전송 정보 목적
AI 모델 배포 서버(예: Hugging Face) 모델 파일 요청 URL, 기기의 IP 주소 Gemma 모델 파일 다운로드
공개 환율 API 통화 코드, 기기의 IP 주소 실시간 환율 조회

위 통신은 표준 HTTPS 를 사용하며, 이용자의 대화·이미지·경비 내역은 절대 포함되지 않습니다.

5. 이용자의 권리와 행사 방법

이용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다.

  • 열람·수정·삭제: 앱 내 "설정 > 데이터 관리" 에서 저장된 대화, 경비 기록을 확인·삭제할 수 있습니다.
  • 전체 삭제: 앱을 기기에서 제거하면 모든 데이터가 즉시 삭제됩니다.
  • 권한 거부·철회: Android 시스템 설정에서 카메라 등 앱 권한을 언제든지 허용/거부할 수 있습니다.

6. 앱 권한 안내

권한 필수/선택 사용 목적
인터넷(INTERNET) 필수 AI 모델 다운로드, 환율 조회
카메라(CAMERA) 선택 메뉴판 촬영 및 OCR
저장소(READ/WRITE_MEDIA_IMAGES) 선택 갤러리에서 메뉴 이미지 선택
네트워크 상태 확인(ACCESS_NETWORK_STATE) 필수 오프라인 여부 확인 및 오류 안내

7. 만 14세 미만 아동의 개인정보

앱은 별도의 회원가입이나 개인정보 수집 기능이 없어 만 14세 미만 아동을 식별하거나 타깃팅하지 않습니다. 보호자는 자녀의 앱 사용 및 권한 허용 여부를 확인해 주시기 바랍니다.

8. 개인정보의 안전성 확보 조치

  • 네트워크 구간 TLS/HTTPS 암호화
  • 기기 내부 저장소에 한정한 데이터 보관
  • 불필요한 로그 및 식별자 미수집
  • 외부 추적·광고 SDK 미포함

9. 개인정보처리방침의 변경

본 방침은 법령 및 앱 기능 변경에 따라 개정될 수 있습니다. 중요한 변경이 있을 경우 앱 내 공지 또는 본 페이지를 통해 최소 7일 이전에 안내드립니다.
