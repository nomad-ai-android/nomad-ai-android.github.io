데이터 안전 (데이터 처리방침)

최종 업데이트: 2026년 4월 17일

본 페이지는 Google Play 스토어의 "데이터 안전(Data safety)" 섹션에 공시된 내용을 이용자가 보다 자세히 이해할 수 있도록 요약한 문서입니다. 전체적인 법적 고지는 개인정보처리방침 을 참고해주세요.

요약: NOMAD AI 는 사용자의 어떠한 개인 데이터도 당사 서버로 수집·전송하지 않습니다. 모든 대화, 이미지, 경비 데이터는 이용자의 Android 기기 내부에만 저장됩니다.

1. 데이터 수집 및 전송 여부

항목 상태
앱이 사용자 데이터를 수집하나요? 아니요 (기기 내부 저장만 발생)
앱이 사용자 데이터를 공유하나요? 아니요
모든 사용자 데이터가 전송 중 암호화되나요? 예 (HTTPS/TLS 사용)
데이터 삭제를 요청할 수 있나요? 예 (앱 내 삭제 또는 앱 제거)

2. 처리되는 데이터 유형 (기기 내 로컬)

아래 데이터는 서비스 기능 제공을 위해 이용자의 기기 내부에서 처리·저장되며, 외부로 전송되지 않습니다.

데이터 유형 처리 목적 선택 여부 기기 외부 전송
메시지(대화 내용) AI 대화, 번역, 통역 기능 제공 필수 없음
사진(메뉴 이미지) 메뉴 OCR 및 텍스트 인식 선택 없음
금융 정보(경비·지출 메모) 지출 자동 기록 및 통계 선택 없음
앱 설정(언어, 모델 선택) 사용 환경 유지 필수 없음

3. 네트워크 통신이 발생하는 경우

다음 두 가지 경우에 한해 최소한의 네트워크 요청이 발생합니다. 이 요청에는 이용자의 대화·이미지·경비 내역이 포함되지 않습니다.

  • AI 모델 다운로드: 앱 최초 설정 시 Gemma 모델 파일을 공개 모델 저장소(예: Hugging Face)로부터 다운로드합니다.
  • 환율 조회: 이용자가 환율 기능을 사용할 때 공개 환율 API 에 통화 코드를 전달합니다.

4. 보안 관행

  • 네트워크 구간은 HTTPS/TLS 로 암호화됩니다.
  • 민감 데이터(대화, 경비, 이미지)는 기기를 벗어나지 않으므로 서버 측 유출 위험이 없습니다.
  • 앱에는 광고·분석·추적용 제3자 SDK 가 포함되어 있지 않습니다.
  • 앱 권한은 최소한의 범위에서만 요청합니다.

5. 이용자 제어 및 데이터 삭제

  • 개별 삭제: 앱 내 "설정 > 데이터 관리" 에서 대화와 경비 기록을 개별 삭제할 수 있습니다.
  • 전체 삭제: 앱을 기기에서 제거하면 기기 내 모든 앱 데이터가 즉시 삭제됩니다.
  • 권한 제어: Android 시스템 설정에서 카메라·저장소 권한을 언제든 허용/거부할 수 있습니다.
